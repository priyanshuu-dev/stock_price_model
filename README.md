# Stock Price Prediction using Machine Learning

This project predicts stock prices using historical market data and machine learning techniques.
It is a beginner-friendly project created for learning and practice purposes.

#Project Overview
The goal of this project is to analyze historical stock price data and build a model that can
predict future stock prices based on past trends.

#Technologies Used
- Python
- Machine Learning

#Dataset Features:
- Date
- Open price
- High price
- Low price
- Close price
- Volume
- GDP growth
- Inflation Rate
 #Requirements
  Python
  Pandas
  Numpy
  scikit-learn
